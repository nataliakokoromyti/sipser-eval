# Exercise 0.6

## Chapter
Chapter 0: Introduction and Preliminaries

## Problem Type
short_answer

## Problem Statement
Let X be the set {1, 2, 3, 4, 5} and Y be the set {6, 7, 8, 9, 10}. The unary function f : X → Y and the binary function g : X ×Y →Y are described in the
following tables.
a. What is the value of f(2)?
b. What are the range and domain of f?
c. What is the value of g(2, 10)?
d. What are the range and domain of g?
e. What is the value of g(4, f(4))?
