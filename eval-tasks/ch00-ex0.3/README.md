# Exercise 0.3

## Chapter
Chapter 0: Introduction and Preliminaries

## Problem Type
short_answer

## Problem Statement
Let A be the set {x, y, z} and B be the set {x, y}.
a. Is A a subset of B?
b. Is B a subset of A?
c. What is A ∪ B?
d. What is A ∩ B?
e. What is A × B?
f. What is the power set of B?
