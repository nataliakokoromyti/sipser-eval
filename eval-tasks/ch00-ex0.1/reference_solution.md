# Solution: Exercise 0.1

## Solution
Step 1:
a) A set of all odd natural numbers. b) A set of all even integers.

Step 2:
c) A set of all even natural numbers. (or) A set of all natural numbers divisible by 2.

Step 3:
d) A set of all natural numbers, divisible by both 2 and 3. (or) A set of all natural multiples of 6.

Step 4:
e) The set of all strings comprising of 0's and 1's and every string is a palindrome.

Step 5:
f) The set of all integers that are equal to one added to that number.
